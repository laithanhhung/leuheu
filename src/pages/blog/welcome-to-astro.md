# Welcome to Astro

Astro is a modern web framework for building fast, content-focused websites. It allows you to build websites using your favorite UI components and libraries.

## Why Astro?

- **Fast by default**: Astro generates static HTML by default, making your site lightning fast
- **Component-based**: Use your favorite UI framework (React, Vue, Svelte, etc.)
- **Content-focused**: Perfect for blogs, marketing sites, and documentation
- **Server-side rendering**: Great performance with SSR when needed

## Getting Started

To create a new Astro project, run:

```bash
npm create astro@latest
```

## Key Features

- **Islands Architecture**: Interactive components are isolated, keeping your site fast
- **Markdown Support**: Write content in Markdown, Astro handles the rest
- **CSS Scoping**: Styles are automatically scoped to components
- **TypeScript Ready**: Full TypeScript support out of the box

Start building amazing websites with Astro today!
